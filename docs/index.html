离线版安装教程

×
  1. 下载 shentie-hegkjgoefobmnaoagodojcaomhmkocmg.crx.zip 文件并解压，得到 .crx 文件
  2. 打开 Chrome 浏览器，在地址栏输入：chrome://extensions
  3. 在右上角打开"开发者模式"开关
  4. 将解压得到的 .crx 文件直接拖拽到扩展程序页面进行安装
  5. 安装完成后，可以关闭"开发者模式"开关

注意事项：

  • 安装过程中请确保"开发者模式"处于开启状态
  • 如果拖拽安装失败，请检查 Chrome 版本是否需要更新
  • 安装完成后建议关闭"开发者模式"以提高浏览器安全性
